# 💬 Chatting Application

The **Chatting Application** is a real-time messaging system developed using **Java Socket Programming** and **Java Swing** for the graphical user interface.  
The application demonstrates **client-server communication**, allowing users to send and receive messages in real time.

---

# 📚 About the Project

This project was built to understand how **network communication works in Java** using sockets.  
It simulates a simple chat system where a **server and client exchange messages through a network connection**.

The project focuses on:

- Java networking using sockets
- Real-time message communication
- GUI development using Java Swing
- Client-server architecture

---

# 🚀 Features

- Real-time communication between client and server
- Graphical user interface using Java Swing
- Message sending and receiving functionality
- Simple and interactive chat interface
- Demonstrates socket-based networking

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/ChaitanyaBhatt147/Chatting_Application.git
cd Chatting_Application
```

---

## 2️⃣ Compile the Java files

```bash
javac *.java
```

---

## 3️⃣ Run the Server

```bash
java Server
```

---

## 4️⃣ Run the Client

Open another terminal and run:

```bash
java Client
```

---

# 🌐 How It Works

1. The **server application starts first** and waits for client connections.
2. The **client connects to the server using sockets**.
3. Once connected, both client and server can **exchange messages in real time**.
